#### Features
- 🔊  Read aloud quotes using on-device Text-to-Speech engine.
- ©️  Copy quotes.
- 📱  Share quotes.
- 🖊️  Minimalist UI.
- 🔔  Set notifications at specified time to get quotes periodically.
- 🌚  Dark mode support.
